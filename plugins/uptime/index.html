What does this plugin do?

The site availability plugin uses the fantastic Down for everyone or just me? website to check whether a given website is online or not and sends you the result.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/uptime/?callback=<your webhook URL>
Method: GET

Output details

Slackbot will inform you of the site's status.
