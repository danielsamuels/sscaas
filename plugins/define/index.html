What does this plugin do?

The dictionary plugin will get the definition of a word and provide you with it's meaning.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/define/?callback=<your webhook URL>
Method: POST

Usage details

/define word
Replace 'define' with your chosen slash command and word with the word you would like defined.

Output details

Name: Dictionary Bot
Emoji: :book: 📖
