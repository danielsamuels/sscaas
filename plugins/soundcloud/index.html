What does this plugin do?

The SoundCloud plugin will search SoundCloud via it's API for a given track name and, if found, embed it into your channel.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/soundcloud/?callback=<your webhook URL>
Method: POST

Output details

Name: Soundcloud Bot
Emoji: :speaker: 🔊
