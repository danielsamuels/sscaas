What does this plugin do?

This plugin lets users flag their content as NSFW and disables auto expanding any links or content the users posts.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/nsfw/?callback=<your webhook URL>
Method: POST

Usage details

/nsfw your message
Replace 'nsfw' with your chosen slash command and content with the content you'd like to flag

Output details

Slackbot will post the content to the channel flagged as NSFW.

Note: This command requires you to add a custom emoji to your Slack instance. A suitable image is available here.
