What does this plugin do?

The developer excuse plugin will get a developer excuse from the brilliant Programmer Excuses website and send it to Slack.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/excuse/?callback=<your webhook URL>
Method: GET

Output details

Name: Developer Excuse
Emoji: :no_entry_sign: ⛔
