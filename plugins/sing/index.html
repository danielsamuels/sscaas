What does this plugin do?

This plugin will make your words look a little more musical.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/sing/?callback=<your webhook URL>
Method: POST

Usage details

/sing word(s)
Replace 'sing' with your chosen slash command and word(s) with what you would like to be 'sung'.

Output details

Name: Sing Bot
Emoji: :musical_score: 🎼
