What does this plugin do?

The Urban Dictionary plugin gets the Urban Dictionary "definition" of a word and posts it to Slack. Note: These messages are not filtered and will most likely contain the sort of language you'd expect from Urban Dictionary.

How does it look?

Configuration details

URL: https://hooks.sscaas.eu/urbandictionary/?callback=<your webhook URL>
Method: GET

Output details

Name: Urban Dictionary Bot
Emoji: :poop: 💩
